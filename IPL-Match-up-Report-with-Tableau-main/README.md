# IPL-Match-up-Report-with-Tableau

## Project Overview

This project analyzes IPL batter vs bowler matchups using advanced cricket analytics and interactive dashboards.

* The analysis focuses on:

* Head-to-head performance

* Venue impact

* Match phase analysis

* Bowling type effectiveness aganist the batter


## Objective

To identify:

* Batter strengths and weaknesses

* Bowler dominance patterns
  
* Batter performances aganist all the bowlers


# Dashboards Included
## 1️ Batter vs Bowler Matchup Report

* Head-to-head summary

* Season-wise trends

* Dismissal patterns

* Run type distribution

## 2️ Venue Performance Analysis

* Strike rate across venues

* Runs vs wickets by stadium

* Venue-based performance variation

## 3️ Phase Wise Analysis

* Powerplay vs Middle vs Death overs

* Economy & bowling SR

* Phase-wise wicket impact

## 4️ Batting vs Bowling Type Analysis (only for Batter Analysis)

* Balls per dismissal

* Batter Performance vs pace & spin

* Bowling-style effectiveness


# Key Insights

Middle overs are the most decisive phase.
Strike rate varies significantly across venues.
Certain bowling styles create higher dismissal risk.
